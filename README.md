# powerbi-global-superstore-dashboard
An interactive Power BI dashboard analyzing Global Superstore sales, regional performance, product insights, and customer segments using Power BI, Power Query, and DAX.
Ye dashboard **GitHub portfolio ke liye bilkul suitable** hai. Agar tum Data Analyst ki job/internship ke liye apply kar rahe ho, to repository sirf files ka collection nahi honi chahiye—usme documentation bhi professional honi chahiye.


## Objectives

```
- Analyze overall sales performance
- Identify top-selling products
- Compare regional sales
- Explore customer segments
- Monitor monthly sales trends
```

---

## KPIs

```

- Total Sales
- Number of Orders
- Average Sales
```

✔ Sales Trend Analysis

## Dataset

```

The project uses the Global Superstore dataset containing:

- Orders
- Customers
- Products
- Categories
- Regions
- Sales
- Profit
```

---

## Insights

```
- Technology generated the highest sales.

- West region achieved the highest revenue.

- Consumer segment contributed over 50% of total sales.

- December recorded the highest monthly sales.

- Office Supplies and Furniture generated comparable revenue.
```

---

## Recommendations

```

- Increase inventory for Technology products.

- Improve marketing in South region.

- Focus promotional campaigns on Consumer customers.

- Analyze seasonal trends to maximize year-end sales.
```
